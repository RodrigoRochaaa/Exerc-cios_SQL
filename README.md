# Lista de Exercícios de SQL

Este repositório contém exercícios praticados durante o curso de SQL do Teo Me Why.

## Estrutura do Repositório

- Os exercícios estão organizados em pastas por tema, incluindo:
  - Comandos básicos e filtros (WHERE)  
  - Agrupamentos e agregações (GROUP BY)  
  - Common Table Expressions (CTEs)  
- Contém também um projeto final, criado como uma Feature Store para fins de prática aplicada.

## Objetivo

- Consolidar conhecimentos em SQL  
- Praticar a criação de queries complexas  
- Organizar dados de forma estruturada e reutilizável, simulando um ambiente de **Data Engineering**
